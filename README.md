### Project Quickstart

Open your terminal, and run the following commands:

```zsh
cd src
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
flask run
```
From now on, just run 

```zsh
cd src
source env/bin/activate
flask run
```
will start the project.
